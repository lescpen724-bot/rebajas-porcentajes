10. Guía didáctica

Guía didáctica

Esta Situación de Aprendizaje está dirigida al alumnado de 6.º de Educación Primaria en el área de Matemáticas.

El proyecto se centra en el trabajo del cálculo de porcentajes aplicado a situaciones reales de compra, como descuentos y ofertas, favoreciendo un aprendizaje significativo y contextualizado.

Relación entre tareas y criterios de evaluación

Las actividades del proyecto están diseñadas para desarrollar los criterios de evaluación del área de Matemáticas, garantizando la coherencia entre el proceso de enseñanza-aprendizaje y la evaluación.

A continuación se muestra la relación entre las tareas, los criterios de evaluación, el tiempo, los agrupamientos y las herramientas utilizadas.

Metodología

Se emplean metodologías activas centradas en el alumnado, como el aprendizaje cooperativo y la resolución de problemas.

Las actividades se plantean a partir de situaciones reales, favoreciendo la participación activa y la construcción del conocimiento.

Se integran herramientas digitales que permiten al alumnado trabajar de forma autónoma, colaborar con sus compañeros y presentar sus resultados de manera visual.

Organización del aula

El proyecto combina diferentes tipos de agrupamiento:

- Trabajo individual.
- Gran grupo.
- Equipos cooperativos con roles definidos.

Esta organización permite adaptar el aprendizaje a las necesidades del alumnado y favorecer la colaboración entre iguales.

Evaluación

La evaluación es continua, formativa y basada en la triangulación de instrumentos.

Se utilizan:

- Rúbrica para el producto final.
- Lista de cotejo para el seguimiento del proceso.
- Diana de autoevaluación del alumnado.

Estos instrumentos permiten valorar tanto el aprendizaje como el proceso seguido.

Orientaciones para el profesorado

Se recomienda adaptar las actividades en función del nivel del alumnado y ofrecer apoyo adicional cuando sea necesario.

En caso de dificultades técnicas, se pueden utilizar alternativas analógicas para garantizar el desarrollo del proyecto.

Es importante fomentar la participación, el trabajo cooperativo y el uso responsable de las herramientas digitales.
